# ☕ Starbucks Beverage Insights Dashboard

An interactive Power BI dashboard analyzing Starbucks' beverage menu — covering calories, caffeine, sugar content, and category-wise distribution — to uncover nutritional and product trends across the menu.

## 📊 Overview

This dashboard explores Starbucks' beverage catalog to answer questions like:
- Which beverage categories are the highest in calories and caffeine?
- How is the menu distributed across categories (Espresso, Frappuccino, Tea, etc.)?
- Which drinks pack the most caffeine?
- How does sugar and calorie content vary by category?

## 🔑 Key Metrics (KPIs)

| Metric | Value |
|---|---|
| Total Beverages Analyzed | 33 |
| Average Sugar | 33g |
| Average Calories | 194 kcal |
| Average Caffeine | 81 mg |

## 🧩 Features

- **Dynamic Filters** – Protein Range slider and Beverage Prep dropdown for on-the-fly slicing
- **Average Calories by Category** – Area chart trend across beverage categories
- **Beverage Category Distribution** – Donut chart showing menu composition (Classic Espresso, Tazo Tea, Frappuccino, Shaken Iced, Signature Espresso)
- **Average Caffeine by Category** – Horizontal bar comparison led by Coffee
- **Top 5 Highest Caffeine Beverages** – Visual callout cards with images for quick reference
- **Starbucks Global Presence** – Comparative bar chart across select beverages

## 🛠️ Tools Used

- **Power BI Desktop** – Data modeling, DAX measures, and visualization
- **Power Query** – Data cleaning and transformation

## 💡 Key Insights

- Smoothies and Frappuccinos top the calorie chart, while brewed Coffee remains the leanest option at just 4 kcal on average.
- Coffee dominates caffeine content (294mg avg) — far ahead of Classic Espresso (122mg) and Frappuccino-based drinks (~100mg).
- Classic Espresso Drinks make up the largest share (7) of the analyzed menu, followed closely by Tazo Tea (5).
- There's a clear inverse pattern between calorie-heavy blended drinks and high-caffeine brewed options — useful for positioning "light" vs "energizing" menu picks.
